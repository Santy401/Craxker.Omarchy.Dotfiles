# Craxker Omarchy Theme

An Omarchy Theme for your Arch Linux / Hyprland setup based on the Craxker
Terminals found in the Fallout video game franchise.

![Craxker Omarchy Theme Screenshot](./preview.png)

# Installation

To install this theme, simply use the ``omarchy-theme-install``   

command: `` omarchy-theme-install https://github.com/signaldirective/craxker-theme ``
